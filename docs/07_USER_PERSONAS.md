# WashGo — User Personas

You are a Senior UX Researcher, Product Manager, and Human-Centered Design Expert.

Using the previous WashGo documentation (Project Foundation, Product Requirements, Design System, Engineering Standards, Frontend Architecture, and Component Library) as context, create the User Personas for WashGo.

Develop realistic personas for:

1. Customer
2. Rider
3. Laundry Shop Owner
4. Administrator

For each persona include:

- Name
- Age
- Occupation
- Background
- Technical Skill Level
- Goals
- Motivations
- Pain Points
- Frustrations
- Behaviors
- Preferred Devices
- Typical Day
- Why They Would Use WashGo

The personas should represent real users in Cambodia, especially Siem Reap.

Use professional UX documentation suitable for both startup product development and university thesis documentation.
